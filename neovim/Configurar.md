Se hace el archivo `~/.config/nvim/init.lua`

La estructura va a así para la configuración
```vim
.
├── init.lua
├── lua
│   ├── config
│   │   ├── autocmds.lua
│   │   ├── keymaps.lua
│   │   ├── lazy.lua
│   │   └── options.lua
│   └── plugins
│       ├── example.lua
│       ├── lsp.config.lua
│       └── themes.lua
```
